# Manasai Stanly J - Portfolio

A production-ready React + Vite multi-page portfolio clone inspired by TD Alexfolio, rebuilt with a custom light theme and identity.

## Tech Stack
- React 18 + Vite 5
- React Router v6
- Tailwind CSS v3
- Framer Motion v11
- React Icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Customization
- Update identity data in `src/data/content.js`.
- Replace `public/profile.jpg` with your own portrait image (must be named `profile.jpg`).
